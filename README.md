# Kaplans SACCO Management System

A comprehensive Savings and Credit Co-operative (SACCO) management system built with Next.js, Prisma, and PostgreSQL.

## 🚀 Features

### For Members
- ✅ User Registration & Authentication
- ✅ Dashboard with Real-time Statistics
- ✅ Monthly Savings Contributions
- ✅ Loan Application with Guarantors
- ✅ Loan Tracking & Management
- ✅ Withdrawal Requests (M-Pesa & Bank Transfer)
- ✅ Financial Reports & Transaction History

### For Admins
- ✅ Admin Dashboard with Analytics
- ✅ Member Management
- ✅ Loan Application Approval/Rejection
- ✅ Withdrawal Request Processing
- ✅ System Reports & Analytics

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js
- **Styling**: Tailwind CSS with Glass-morphism
- **Icons**: Lucide React

## 📋 Prerequisites

- Node.js 18+
- PostgreSQL 15+
- npm or yarn
